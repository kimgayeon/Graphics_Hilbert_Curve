1. offset 
 - offset은 그룹안의 사각형들 사이의 거리로 설정하였습니다.
 - default값은 0.04입니다.

2. min/max
 - min/max값은 첫번째 level에 그려질 수 있는 사각형 그룹의 최대 최소개수입니다.
 - default값은 min = 15, max = 20입니다.

3. ratio
 - ratio는 각 level에 그리는 사각형 그룹의 비율입니다.
 - default값은 1입니다.

* min/max값을 20/30으로 설정하고 ratio를 2로 설정한 상태에서 모든 level에 min/max를 적용한다면 첫번째 값을 제외하고 모두 30씩 찍히므로 min/max값은 첫번째 단계에서만 적용하였습니다. (그림이 더 풍성하게 보이도록 하기 위해서 입니다!) 

* hilbert curve의 level은 7로 설정하였습니다.
